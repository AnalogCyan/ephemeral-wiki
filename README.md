# Ephemeral Wiki

Every page on this site is generated by AI on each visit and only exists while being viewed. Visit any URL path and get a unique page about that topic.

Live at: https://ephemeral-wiki.onrender.com/

Inspired by [LiveOverflow's video](https://youtu.be/M2uH6HnodlM) and [repo](https://github.com/LiveOverflow/everything-api).

## Stack

- **FastAPI** + **Uvicorn** for the web server
- **OpenRouter** for AI inference (any model, defaults to `openrouter/auto`)
- **Render** for hosting (free tier)

## Local Development

```bash
git clone https://github.com/AnalogCyan/ephemeral-wiki.git
cd ephemeral-wiki
pip install -e ".[dev]"
```

Set your OpenRouter API key:

```bash
export OPENROUTER_API_KEY="your-key-here"
```

Run the server:

```bash
uvicorn app:app --reload
```

Visit http://localhost:8000 to see it in action.

## Configuration

All configuration is via environment variables:

| Variable | Default | Description |
|---|---|---|
| `OPENROUTER_API_KEY` | (required) | Your OpenRouter API key |
| `MODEL` | `openrouter/auto` | Model to use for generation |
| `RATE_LIMIT` | `10/minute` | Per-IP rate limit |
| `CACHE_TTL` | `600` | Cache lifetime in seconds |

## Deployment

The app is configured for Render via `render.yaml`. Connect your GitHub repo in the Render dashboard and it will auto-deploy. Set `OPENROUTER_API_KEY` in the Render environment variables.
